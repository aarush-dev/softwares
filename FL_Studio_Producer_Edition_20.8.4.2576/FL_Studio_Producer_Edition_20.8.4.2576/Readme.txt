Для установки запустите FL Studio 20.8.4.2576 Setup.exe
В раздаче используется официальный установщик FL Studio 20.8.4.2576.
После завершения работы официального установщика происходит автоматическая активация FL Studio.
Важно! Если, во время установки FL Stuido, официальный установщик попросит выполнить перезагрузку - откажитесь, выполнив ее позднее.

Дополнительные плагины из версии FL Studio 20.9.1.
==================================================
В папке "Addition Plugins from version 20.9.1" находится установщик дополнительных плагинов Pitch Shifter и Vintage Chorus от версии 20.9.1.
Плагины устанавливаются только для версии 20.8.4.2576 и будут работать только с вылеченными файлами FLEngine в 64-битной версии FL Studio.
Если в системе установлено несколько версий FL Studio, то FL Studio 20.8.4.2576 Plugins Setup.exe нужно запускать из папки с версией 20.8.4.2576

Внимание! В официальной версии FL Studio 20.8.4 этих плагинов нет, устанавливая их вы осознаете все риски использования данных плагинов в проектах.

==================================================
Дополнительная информация:
==================================================
Отключена отправка данных на сервера разработчиков в FL Studio и FLEX (Поэтому обложки для расширений FLEX не будут видны).
Плагины, требующие подключения к сети, будут работать.
Из FLEngine удален блэк-лист заблокированных пользователей.
Разблокированы триальные плагины.

Patch:
	В папке Patch находится патч-активатор для FL Studio версии 20.8.4.2576.
	Если в системе установлено несколько версий FL Studio, то FL Studio 20.8.4.2576 Patch.exe нужно запускать из папки с версией 20.8.4.2576.
